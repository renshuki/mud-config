# mud-config
My tt++ configuration for playing MUDs

## Install 

+ Git clone the repo

> You need TinTin++ installed and executable via `tt++` command

## Usage 

+ Execute `./run`
> Verify permissions are set to 700

## Macros

### Ensimud  

Load character's actions, aliases, substitues and session.

| Key | Character | Class |
| --- | --------- | ----- |
| F1 | Zephir | Paladin |
| F2 | Siryus	| Priest |
| F3 | Shiru | Magician |
| F4 | Suki | Warrior |
